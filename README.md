--- Real-Time Restaurant Ordering System ----

A mini restaurant management solution featuring a customer-facing menu application and a kitchen dashboard for order management. Customers can browse menus, add items to cart, and place orders from their table. Kitchen staff receive orders in real-time and can manage order status with instant customer notifications.

## Features

### Customer App
- 📱 Browse menu by categories
- 🛒 Add items to cart with quantity control
- 💳 View order summary with taxes and service fees
- 🔔 Real-time notifications when order is ready
- 📍 Table-based ordering system

### Kitchen Dashboard
- 📊 View incoming orders in real-time
- ✅ Order management with status tracking
- 🍽️ Send order completion notifications to customers
- 📝 Order history tracking
- 📦 Inventory management system

## Tech Stack
- **Frontend:** React.js, React Router
- **Backend:** Firebase/Firestore (real-time database)
- **Styling:** Tailwind CSS
- **Icons:** React Icons, Lucide React
- **Deployment:** Vercel

## Key Highlights
- ⚡ Real-time order synchronization
- 🔊 Audio notifications for order completion
